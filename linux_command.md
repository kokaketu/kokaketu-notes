# Linux常用命令
### 一、基础命令
1. 帮助命令
	```
	# useradd -h
	```
2. 切换目录

  	切换到当前目录
	```
	# cd .
	```
  	切换到上一级目录
	```
	# cd ..
	```
	列出当前目录下的文件
	```
	# dir
	```
	```
	# ls
	```

   用列表的方式列出文件
	```
	# ls -l
	# ll
	```

### 二、用户相关命令

1. 新增用户

	```
	# useradd kokaketu
	```

2. 修改用户密码

	```
	# passwd kokaketu
	Changing password for user kokaketu.
	New password: 
	Retype new password:
	```
